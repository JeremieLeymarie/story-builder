## How to use Server

# Path to do All

- login

```
/api/login
```

- register

```
/api/register
```

# Path to do builder

- Save your game data

```
/api/builder/save/game
```

- Load your game data

```
/api/builder/load/game
```

# Path to do player

- Save your game data

```
/api/player/save/game
```

- Load your game data

```
/api/player/load/game
```
